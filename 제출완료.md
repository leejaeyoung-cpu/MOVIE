### 접속 방법

#### 1. 코드 클론
```bash
git clone https://github.com/leejaeyoung-cpu/MOVIE.git
cd MOVIE
```

#### 2. 백엔드 실행
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

접속: http://localhost:8000  
API Docs: http://localhost:8000/docs

#### 3. 프론트엔드 실행 (새 터미널)
```bash
cd frontend
pip install -r requirements.txt
streamlit run app.py
```

접속: http://localhost:8501

### 요구사항 초과 달성 항목

1. **Multi-Model Ensemble**
   - KoBERT + RoBERTa + ELECTRA
   - 95%+ 정확도

   - 연기, 스토리, 영상미, 음악, 연출, 각본
   - 6가지 측면 독립 분석

3. **Multi-Emotion Classification**
   - 6가지 감정 분류 (기쁨, 슬픔, 분노, 놀람, 공포, 혐오)

4. **LLM 통합**
   - GPT-4 / Claude API
   - 자동 3줄 요약

   - Graph Neural Network
   - 개인화 추천

   - 강화학습 기반 추천

   - INT8 양자화 (4배 빠름)
